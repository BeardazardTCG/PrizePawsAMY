# Bark Booth

Celebrate every milestone.

Bark Booth is a frontend-only concept for a UK-based dog community centred around competitions, achievements, fair judging, and positive owner-dog experiences. It is designed to feel like a digital scrapbook of a dog's achievements, with mock competitions, dog profiles, badges, rosettes, winners, and mobile-first navigation.

This build intentionally uses static mock data only. There is no auth, database, payments, upload storage, API integration, or real user system.
